# SlimeHUD

添加了 Slimefun 方块的 WAILA (我在看什么) 显示功能。  
你可以直接获得你看向的方块的名称，不需要打开菜单，或是破坏方块。  

可以在`config.yml`中设置使用BossBar或ActionBar显示。  
额外的能源/货运网络信息也可以设置开启或关闭。

玩家可以使用指令 `/slimehud toggle` 来切换 WAILA 的开关。
